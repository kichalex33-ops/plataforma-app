# Seguradora, Guincho e Assistência

## Objetivo

Preparar a comunicação futura com seguradora, guincho e assistência técnica para ocorrências de frota.

## Eventos Candidatos

- Pane mecânica.
- Pneu furado.
- Acidente.
- Manutenção emergencial.
- Solicitação de guincho.
- Bloqueio operacional do veículo.

## Dados Mínimos

- Viagem.
- Veículo.
- Placa.
- Motorista.
- Localização quando disponível.
- Tipo da ocorrência.
- Descrição.
- Fotos.
- Checklist relacionado.

Existe fila simulada e log interno. Nenhum endpoint real é usado.
