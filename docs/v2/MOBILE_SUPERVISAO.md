# Mobile Supervisão

## Objetivo

Planejar recursos mobile para supervisores em campo.

## Funcionalidades Futuras

- Lista de viagens por status.
- Filtro por motorista e veículo.
- Alertas operacionais.
- Aprovação de revisão.
- Acompanhamento de ocorrências.
- Indicadores compactos.

## Regras

- Não misturar visão do motorista com visão de supervisor.
- Permissões devem controlar ações sensíveis.
- Dados críticos precisam de auditoria.
