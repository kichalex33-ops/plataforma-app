# Inteligência Operacional

## Objetivo

Planejar uso de IA para apoiar análise operacional sem substituir decisão humana.

## Casos Candidatos

- Resumo automático de viagem.
- Detecção de inconsistências.
- Sugestão de revisão por KM divergente.
- Análise de atrasos recorrentes.
- Relatórios executivos.

## Regras

- IA deve explicar a origem dos dados usados.
- Decisões críticas exigem confirmação humana.
- O app deve ter fallback mockado ou local quando API futura não existir.
