# WhatsApp Operacional

## Objetivo

Planejar comunicação operacional futura por WhatsApp, sem substituir registros oficiais do app.

## Usos Candidatos

- Aviso ao motorista.
- Confirmação de viagem.
- Notificação de atraso.
- Mensagem para paciente.
- Canal de contingência.

## Regras

- Toda comunicação importante deve gerar registro no sistema.
- WhatsApp deve ser apoio, não banco oficial.
- Dados sensíveis devem ser minimizados.
