# Roadmap 2.0

## Objetivo

Planejar a evolução da demo para um app único real, sem reescrever agora os módulos preservados.

## Etapas

1. Estabilizar demo mobile atual.
2. Consolidar Logística operacional offline-first.
3. Preparar base modular para expansões futuras quando houver novo escopo aprovado.
4. Criar camada real de permissões por perfil.
5. Preparar sincronização com backend futuro.
6. Adicionar app de supervisão e tempo real.
7. Integrar BI, inteligência operacional e serviços externos.

## Critérios

- App precisa continuar compilável.
- Módulos existentes não devem ser quebrados.
- Novas integrações devem funcionar com fallback local.
- Funcionalidades críticas precisam de testes.
