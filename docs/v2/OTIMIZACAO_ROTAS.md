# Otimização de Rotas

## Objetivo

Planejar melhoria futura de rotas para reduzir tempo, custo e deslocamentos desnecessários.

## Dados Necessários

- Endereços de embarque.
- Unidade de destino.
- Horários de consulta.
- Prioridade.
- Acessibilidade.
- Capacidade do veículo.

## Cuidados

- Rota sugerida não deve ignorar regras de saúde.
- Motorista deve poder abrir navegação externa.
- Registro final deve refletir o que realmente ocorreu.
