# Multi-Município

## Objetivo

Planejar operação para múltiplos municípios usando a mesma base tecnológica.

## Requisitos

- Separação de dados por município.
- Perfis e permissões por município.
- Configurações locais.
- Identificação clara no app.
- Relatórios filtrados.

## Riscos

- Vazamento de dados entre municípios.
- Configurações conflitantes.
- Crescimento de volume sem paginação e sincronização incremental.
