Projeto: Controle ACE Territorial

Stack:
- Flutter
- SQLite
- Android

Objetivo:
Sistema territorial para Agentes de Combate às Endemias.

Módulos:
- Pontos Estratégicos
- BTI
- Ovitrampas
- Mapa epidemiológico
- Dashboard
- GPS
- Fotos
- Relatórios PDF

Offline-first com SQLite.

- Status automático em desenvolvimento
